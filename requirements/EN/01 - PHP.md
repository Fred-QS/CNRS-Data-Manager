# PHP

![PHP 8.1 logo](/documentation/attachments/PHP8.1.jpg)

The minimum PHP version required is [version 8.1](https://www.php.net/releases/8.1/en.php). The libraries used and the security necessary for the proper functioning of the extension are the main reasons.

![PHP Performances scheme](/documentation/attachments/PHP-Perf.png)

As WordPress applies backward compatibility, this, for security reasons, unfortunately cannot be applied to the CNRS Data Manager extension.
